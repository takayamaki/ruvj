## 開発ルール

JSONの単純なパースには **`jq`** を使うこと。
単純なパースを超えた複雑な処理を行うときは node を使ってもよい。
同じ処理を3回繰り返した場合、以降も行う可能性があるならば `.tmp/` ディレクトリに
ユーティリティスクリプトとして書き出して使うこと。
行数の多い実行結果が予想されるときはリダイレクトで `.tmp/` に書き出してから読み込むこと。

skillの実装規約は [build-skill.md](rules/build-skill.md) を参照すること。

## git 操作

`git` の直接呼び出しは禁止。必ず `git-agent` を使うこと。
